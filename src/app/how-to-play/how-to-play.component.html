How to Play?

STEP #1
(KNOWLEDGE OF KAPAMPANGAN VOCABULARY)

If "Yes" is selected, you will have access to the SELECT MODE Screen. Otherwise, only REVIEW FLIPCARDS with basic Kapampangan words/phrases translation will be accessible.

STEP #2 (SELECT A MODE)

If BEST RESPONSE is chosen, you will answer a given scenario based on a category, see how others responded, and learn the most appropriate answer.

If REVIEW FLIPCARDS is selected, you can familiarize yourself with the Kapampangan words/phrases translation used in the scenarios that you still don't know.
