How to Play?

STEP #1


If "Yes" is selected, the user has access to both the Best Response Game and Review Flipcards.





If "No" is selected, only Review Flipcards with Basic Words will be accessible.

STEP #2


If "Yes" is selected, the user must choose which mode to play.








If "No" is selected, only Review Flipcards with Basic Words will be accessible.

STEP #3


If "Yes" is selected and the Best Response Game is chosen, the user will answer a given scenario, see how others responded, and learn the appropriate answer. If the user chooses Review Flipcards, they can read and become familiar with the words used in the scenarios.



If you selected Review Flip Cards: Now you can read and be familiar with the Kapampangan words that you still don't know.
